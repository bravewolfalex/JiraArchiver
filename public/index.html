Jira Archiver

Export Jira issues to HTML files based on JQL queries

Enter your Jira instance URL (e.g., https://yourcompany.atlassian.net)
Copy the cookie from your browser's developer tools while logged into Jira
Enter a JQL (Jira Query Language) query to filter issues
Example JQL Queries:
project = MYPROJECT AND status = Done
assignee = currentUser() AND created >= -30d
project = PROJ AND priority = High
text ~ "bug" AND resolved >= -7d
